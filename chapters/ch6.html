CHAPTER 6

예측·분류 모델 mAb-GATED

그래프에서 안정성 노드 하나를 가리고, 이웃 맥락만으로 맞히는 모델을 학습시킵니다.

이 장의 결과물 학습된 mAb-GATED 모델 + 성능표(정본 초록: Test n=1,231, Hits@1 84.7%, MRR 0.884). 베이스라인을 크게 앞섭니다. (GPU 런타임 권장)
🎯 직접 눌러 확인하기 안정성 노드를 가리고 맞히는 과정을 mab-gated-demo.vercel.app에서 실제 지식그래프로 직접 해볼 수 있습니다(‘안정성 예측 검증’ 탭).

1. 직관 — “빈칸 채우기”로 패턴을 배운다

그래프에서 안정성 노드 하나를 가리고 “이웃들을 보니 여기 들어갈 단어는?”을 맞히게 합니다. 문헌에 반복된 인과 패턴이 정말 ‘학습 가능한가’를 시험하는 거죠.

‘같은 범주 안에서’ 맞히기 아무 노드나 후보로 두면 너무 쉽습니다. 그래서 같은 하위범주(물리·화학·생물) 안에서 정답을 1등으로 올리도록 합니다. 예측 대상은 32개 표준 타깃을 15개 공식 지표(물리 4·화학 5·생물 6)로 정리한 것입니다.

2. 이름 풀이와 구조

이름 GATED = Graph-Aware Transformer with Encoder-Decoder. 네 부품이 순서대로 이어집니다(총 약 320만(3.2M) 파라미터, 임베딩 차원 D=128).

1

임베딩 — PubMedBERT로 시작

각 노드 이름을 생의학 언어모델로 768차원 → 128차원으로 투영해 초기화. 의미가 가까운 노드는 가까운 자리에서 출발합니다.

2

GAT — 그래프 맥락 주입

전체 지식그래프(8,670 노드·32,939 엣지, 정·역방향)를 어텐션으로 이웃 정보를 모음.

3

Transformer Encoder

입력 토큰들끼리 self-attention → 평균 풀링으로 맥락 벡터(128차원).

4

Transformer Decoder

cross-attention으로 예측 벡터 생성 → 후보 노드별 점수(logits) 출력.

PubMedBERT 초기화가 실제로 의미 있다 초기 임베딩의 코사인 유사도를 보면 — conformational stability ↔ thermal stability 0.914, sucrose ↔ trehalose 0.897, oxidation ↔ deamidation 0.850. 의미가 통하는 노드끼리 이미 가까이 모여 있습니다.

3. 입력 토큰 — 한 토큰에 4가지 정보

입력은 T=30 토큰(6개 도메인 카테고리 × 카테고리당 이웃 K=5). 각 토큰은 원인 노드 하나를 네 측면으로 인코딩합니다.

  • 노드 ID — 어떤 노드인가
  • 카테고리 — 6색 도메인 중 무엇인가
  • 관계 유형 — promotes / stabilizes 등
  • 인과 거리 — 직접(1-hop)인가, ‘원인의 원인’(2-hop)인가

4. 데이터셋과 베이스라인

  • 분할: train 70% / valid 15% / test 15% (계층 추출).
  • 학습: 하위범주로 제한한 cross-entropy, Adam(lr=1e-3), epoch 59에서 early stop(best valid MRR 0.8909).
  • 겨룰 상대: Random(하한) · Frequency(최빈 노드 찍기) · DistMult(고전 KG 임베딩).

5. 핵심 코드 (개념)

cell · model (발췌)
import torch, torch.nn as nn

class mAbGATED(nn.Module):
    def __init__(self, num_nodes, D=128, heads=4):
        super().__init__()
        # 4개 임베딩: 노드ID / 카테고리 / 관계 / 인과거리
        self.node_emb = nn.Embedding(num_nodes+1, D, padding_idx=0)
        self.gat      = GATLayer(D, D, n_heads=heads)        # 그래프 맥락
        enc = nn.TransformerEncoderLayer(D, heads, batch_first=True)
        self.encoder  = nn.TransformerEncoder(enc, num_layers=2)
        dec = nn.TransformerDecoderLayer(D, heads, batch_first=True)
        self.decoder  = nn.TransformerDecoder(dec, num_layers=2)
        self.out      = nn.Sequential(nn.LayerNorm(D), nn.Linear(D, D),
                                      nn.GELU(), nn.Dropout(0.1), nn.Linear(D, num_nodes+1))

    def forward(self, tokens, edge_index):
        x = self.node_emb(tokens)
        x = self.gat(x, edge_index)              # 90,450개 엣지로 메시지 전달
        ctx = self.encoder(x).mean(dim=1)        # 맥락 벡터(128)
        q   = self.decoder(x[:, :1], ctx.unsqueeze(1))
        return self.out(q.squeeze(1))            # 후보 노드 점수

# 노드 임베딩은 PubMedBERT(pritamdeka/S-PubMedBert-MS-MARCO)로 초기화
평가 지표 MRR(정답 순위의 역수 평균)과 Hits@k(상위 k 안에 정답이 든 비율). 1에 가까울수록 좋습니다.

7. 추출 모델 버전에 따른 차이

추출에 쓴 생성형 AI 모델(GPT-4o-mini ↔ GPT-5-mini)에 따라 엣지 수·관계 분포·성능이 달라집니다. 정본 시스템은 정밀도 우선(precision-first) 필터로 5,576편 → 32,939 엣지(8,670 노드)를 사용합니다. 두 버전의 단계별 비교표는 병합 논문 Appendix A에 정리돼 있습니다.
