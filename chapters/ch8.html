# Abstract
Motivation: 단일클론항체의 IV→SC 전환에서 응집·점도·산화 같은 안정성 지표가
  공유 메커니즘으로 얽혀 있으나, 대개 개별적으로만 연구되어 왔다. ...

  16,019개 엣지의 기전 지식그래프를 구축했다. 허브 노드는 aggregation·stability·

# Key Messages
- 문헌에서 생성형 AI로 mAb 안정성 기전 지식그래프를 대규모 자동 구축.
- PubMedBERT+GAT 기반 mAb-GATED로 안정성 노드 예측 H@1 ≈ 0.80(물리 MRR 0.995).
- 미관측 인과 후보를 검증 가능한 가설로 제시 · GPT-5-mini 재실행으로 추가 향상.

# Methods 소제목 → 각 장 매핑
- Data Collection (PubMed E-utilities, 126 queries / 7 categories)  ← 1장
- Relevance Filtering (GPT-4o-mini)                                 ← 2장
- Extraction + Normalization + Aggregation (159 canonical terms)   ← 3장
- mAb-GATED (PubMedBERT init → GAT → Transformer enc/dec)          ← 6장
- Human Validation (적절율) / Evaluation (MRR, Hits@k)             ← 5·6장
