CHAPTER 7

새로운 가설 만들기

모델이 “있을 법하다”고 보지만 아직 문헌엔 없는 인과 후보를 찾아, 검증 가능한 가설로 정리합니다.

이 장의 결과물 가설 후보 표 — (원인 → 결과, 예측 점수, 카테고리, 근거 이웃, 문헌 부재 확인). 8장 논문의 ‘새로운 발견’ 절이 됩니다.
여기서부터는 ‘재현’이 아니라 ‘창작’ 1~6장은 만들어진 길을 따라왔다면, 이제 모델로 아무도 정리하지 않은 지식 후보를 제안합니다. 이 책의 클라이맥스입니다.

1. 아이디어 — 빈칸은 곧 가설

6장 모델은 “이웃을 보고 가운데 노드”를 맞힙니다. 이 능력을 뒤집으면, 아직 연결되지 않은 두 노드 사이에 화살표가 있어야 할 확률을 점수로 매길 수 있습니다. 점수는 높은데 실제 데이터엔 없는 쌍 = 유망한 가설입니다. 이를 link prediction이라 합니다.

2. 미관측 쌍에 점수 매기기

현재 그래프에 없는 (원인, 결과) 쌍 전체에 모델 점수를 매깁니다. 단, 말이 되는 방향만 — 카테고리 제약을 둡니다(예: stress → stability는 가능, 그 반대는 제외).

cell · score unobserved (개념)
observed = set(zip(df.cause, df.effect))     # 이미 문헌에 있는 쌍
candidates = []
for c in cause_nodes:
    for e in stability_nodes:
        if (c, e) in observed: continue         # 새 후보만
        if not allowed_direction(cat[c], cat[e]): continue  # 카테고리 제약
        score = model.score(c, e)                # 6장 모델로 점수화
        candidates.append((c, e, score))

top = sorted(candidates, key=lambda x: -x[2])[:30]   # 상위 30개

3. 상위 후보 → 문헌으로 교차 확인

모델이 높게 본 후보가 정말 문헌에 없는지 PubMed로 한 번 더 확인합니다(1장의 검색을 재사용). 이미 보고된 관계면 ‘발견’이 아니라 ‘검증’일 뿐이니 걸러냅니다.

cell · novelty check
for c, e, s in top:
    _, n_hits = search_pubmed(f'"{c}" AND "{e}"', max_results=1)   # 1장 함수
    novelty = "신규 후보" if n_hits == 0 else f"기존 {n_hits}건"
    print(f"{c:25s} → {e:20s}  score={s:.3f}  {novelty}")

4. 가설 카드로 정리

숫자만으로는 가설이 아닙니다. 각 후보를 검증 가능한 문장으로 다듬습니다.

가설 카드 (예시)

  • 가설: arginine 농도를 높이면 self-association 감소를 매개로 점도가 낮아질 것이다.
  • 근거 이웃: arginine ⊣ self-association, self-association → viscosity (두 엣지가 사슬을 이룸)
  • 모델 점수: 0.86 (상위 3%)
  • 문헌 상태: 직접 연결 보고 없음(신규 후보)
  • 검증 방법: arginine 0–150 mM 제형에서 점도·DLS 측정

5. 우선순위화 — 무엇을 먼저 시험할까

후보가 많으면 임팩트 × 검증 난이도로 정렬해, 실제로 해볼 만한 1~3개를 고릅니다.

기준질문
임팩트맞다면 SC 제형 개발에 실제로 도움이 되나?
검증 난이도기존 장비·시료로 빠르게 확인 가능한가?
신규성정말 아무도 직접 보고하지 않았나?
기전 타당성이웃 사슬로 말이 되는 메커니즘이 그려지나?
예측 ≠ 사실 모델 점수가 높다고 참은 아닙니다. 가설은 실험으로 반증 가능해야 가치가 있습니다. 이 겸손한 태도가 8장 Discussion(한계)의 핵심입니다.

오늘의 마무리

  • link prediction으로 미관측 인과 쌍에 점수를 매겼습니다.
  • 문헌 대조로 진짜 신규 후보를 걸러, 가설 카드로 정리했습니다.
  • 임팩트·난이도로 우선순위를 정했습니다.

이제 1~7장의 모든 결과가 손에 있습니다. 8장에서 이를 한 편의 논문으로 엮습니다.
